# -Realm-Defense-s-Guide-
Realm Defense's Guide for Beginners
  对于帝国保卫战(英文名 [RD] Realm Defense TD<tower defense> 塔防)这个游戏的很多不想花钱玩的新手玩家来说, 时常会问到的问题是该买什么英雄, 该觉醒哪个英雄和该怎么开图？即使是没想过这些问题的玩家也应该好好想想, 因为这款游戏想有一个相对快速和轻松的经验是需要玩家很好的利用游戏里无论是过图还是其他方式得到的资源都能好好的利用, 否则在某个点上, 玩家会发现有些地图很难通关, 或者是通关太过吃力, 往往需要消耗大堆的道具. 但理论上来讲每世界三到世界六合起来都是不需要消耗超过6颗陨石的. 合理的塔防布置, 合理的英雄选择, 合理的英雄觉醒以及合理的英雄放置, 甚至合理的英雄微操作是完全可以达到0道具的使用. 甚至技术够高的话, 用一些弱的英雄都能通关一些其他玩家用高阶英雄都有一定难度攻克的关, 只要你有相应的技术. 对于帝国保卫战来说, 玩法绝对不是一成不变的, 只要合理的利用资源, 一切皆有可能. 
以下的攻略以及小贴士是基于很多fandom还有reddit论坛上的帖子以及一些本人的经验所完成的关于如何将资源最大化利用的办法. 本篇攻略主要分为四个部分. 读者可根据自己的需求直接看自己想看的部分. 以下是目录.

1.首先是介绍帝国守卫最重要的几种物资
1.钻石: 游戏里的主要货币, 可通过游戏内不同的方式以及充值获得. 是买英雄以及觉醒英雄的必须品, 如何运用钻石将决定你游戏进度的快慢以及体验. 这里有个其他玩家那边得来的例子, 假如你买了3200钻石的礼包, 合理的运用它们将让你的游戏进度快至少一个月. 游戏内获得钻石的方式将在资源介绍完成后详细的解释.


2.圣水: 圣水的唯一用途就是用来升级英雄和合成道具. 由于合成道具需要消耗圣水, 所以建议玩家合成一定数量的道具够用就好, 不要全部把道具合成完了. 圣水也是一种很重要的资源, 可以通过升级圣水工厂让圣水产量加倍. 但是圣水到有大需求的时候是到了游戏大后期要打锦标赛的时候了, 所以是不建议玩家前期花费太多的钻石升级泉水, 价值完全不等, 严重拖慢了推图的进度. 前期只要用少量的泉水将新购买的英雄提升到一个适合能用的等级就可, 剩下的靠过图升级英雄即可, 剩下的圣水要攒起来需要投资到下一个新买的英雄身上. 这也是为什么圣水的使用需要精打细算. 圣水的获取方式也将在资源介绍完成后详细的解释.


3.道具: 游戏里主要有霜冻药剂, 治愈药剂, 火焰药剂, 召唤药剂, 金袋, 毁天灭地魔咒和陨星这7种道具. 关于每种道具的详细解释请自行查看游戏内挑战世界地图2开启的铁匠铺里的信息. 理论上道具是不应该在推图的过程中用到, 所有的道具都应该使用在锦标赛里, 当然这是最理想的情况下, 但是重要的点是, 千万不要浪费道具. 道具将是锦标赛最重要的战略物资, 而锦标赛决定了玩家解锁世界6地图的速度. 每个道具对于哪个段位的锦标赛将在以下锦标赛的攻略得到更详细的解释. 道具的获取方式也将在资源介绍完成后详细的解释.


4.经验值： 经验值其实是另一种形态的圣水, 或者说圣水就是另一种形态的经验值. 根据QQ名叫做1的群友提供的数据, 每个世界的关卡通关一星二星或者三星有不同的经验值奖励. 假如玩家1第一次通关为1星, 第二次通关3星的话将会为玩家1补齐3星跟1星中间的经验值差额. 总体来说就是一次性3星通关和分两次三星通关的经验值奖励是一样的. (希望有热心的群友能帮忙验证下分两次三星和一次性三星通关经验值有差额么？提供数据的群友我也会在攻略里提到你的名字）话归正题, 我希望大家每次通关的尽量三星通关, 不提其中的星星可以差别可以更快的升级科技树, 后者的道具和经验奖励也是不可忽视的物资. 经验值的唯一获取途径就是通关卡的普通模式和传奇模式, 所以经验值的获取是非常有限的. 尽快拿到新英雄然后尽快换上新英雄获取经验值也能帮助节省泉水, 让更多得以攒下来在以后使用. 


注: 笔者发现似乎召唤药剂和霜冻药剂是最稀少的道具, 所以希望群友们除了世界1第20关的时候尽量不要单独使用这两道具, 2图一般流星, 三图以上就是使用陨石了. 建议大家唯一需要使用的道具就是流星和陨石了. 当然大家有大家的玩法, 或许你觉得单独使用某些道具更节省道具也说不定？


2.各种物资的获取途径
1.钻石获取方法
境界守卫每10个徽章打开一次宝箱, 一次宝箱开出50钻石.（注: 对于新手来说只有个别地图加5难度是及其容易的,其他地图尤其是世界以上的地图都相对较难, 新手通关世界2后境界守卫里的只有世界1和2地图, 所以新手玩家可以抓住这个短暂的缓冲期尽力打加5难度的境界守卫）
30天登录奖励会给予一定的钻石, 不过30天后就没有了.
无限模式是世界1到5的一种战役模式. 在一种本世界的地图上有无数波的敌人来袭, 而玩家的任务就是撑过2分钟30秒到15分钟, 有五种时间段分别给予40, 90, 150, 200, 250的钻石.
日常过图普通模式和传奇模式都会给予很少量的钻石.
完成游戏内的任务将给予相对较多的钻石
每日试练应该是玩家最稳定并且最大量钻石的来源了, 尤其是在第三十天的那个周期.
成就系统是一个很容易被忽视的钻石来源, 也因为很多成就难以完成. 新手玩家做每日试练超过两月后会有一个看广告然后跳过战斗的选择, 希望通过成就来解锁成就获取更多钻石的玩家请不要跳过战斗, 因为将很大的帮助玩家完成很多困难的成就. 另外还有一些隐藏的成就, 隐藏的成就列表就在群文件里, 请大家自行查看. 大家千万别忽略完成成就所能得到的钻石.
锦标赛对于新手来说是基本可以忽略的钻的来源地, 因为不是所有玩家都跟你是从同一个时间开始玩的, 所以竞赛名次上不了前10别气馁, 这篇攻略就是为了尽快缩短新玩家和老玩家差距所准备的. 加快推图也能让新手对于其他的新手玩家来说在锦标赛保有相应的优势以得到更高的名次从而更快的推图. 打个比方, 假如说你推图比人家快不少到了110关, 你买了紫龙而人家没有, 你在锦标赛的优势就很大了因为紫龙在锦标赛是作用很大的一个英雄. 所以推图快就是加快获取资源的速度, 而获取资源快又能增加你在锦标赛的竞争力, 而锦标赛的竞争力又能增加你推图的速度. 一切都是相辅相成的. 要知道最后一个级别的锦标赛的第一名可是能奖励500钻的. 不说传说赛的第一名, 即使是钻石赛的第一名的奖励也有300钻. 
幸运大转盘是基本可以忽略不计的获取钻石的来源, 我运气好两月左右拿到过三百钻好像. 不过不要白不要不是.
隐藏关卡可以拿到极少量的钻石（新手极度不推荐）
2.圣水获取方法
升级圣水工厂（新手玩家不推荐花费太多的钻石在升级圣水工厂上）
时空裂缝（请以最高的分通关以得到最高的泉水奖励, 如果第一次不是最高的分通关没关系,回头可以补）
30天登录以及幸运大转盘
锦标赛的名次对于新手来说难以获得, 这个来源基本忽略不计
3.道具获取方法
30天登录, 游戏厅, 幸运大转盘, 通普通模式的关卡（看广告得双倍道具）
看广告拿卡片, 商店里免费每天领取两把钥匙. 然后用钥匙打无限模式, 无限模式会根据玩家撑住的时间给予奖励, 之后可通过卡片和钥匙一起可以跳过战斗直接领取奖励.
能日常登录脸谱网的玩家可以找个号或者另外个玩家互相送礼物, 每个礼物会随机给四种药剂的其中一种.
注： 所有的资源都是及其有限的, 任务只有那么多, 成就也只有那么多, 每张地图的资源也只有那么多. 而道具的消耗在越高等级的锦标赛越是巨大, 所以请小心运用自己所拥有的资源.


3.解锁世界3到世界6
1.前言---请看下面部分之前先看下萌新入群必看文件夹里的<英雄介绍>
2.关于购买哪些英雄以及觉醒哪些英雄的思路的重点
不要因为可以开启隐藏关就购买对推图毫无帮助的英雄, 收益和付出完全不成正比. 极度的浪费资源
购买英雄主要考虑的是对于推图的帮助以及其次的对于锦标赛的帮助
觉醒哪个英雄必须考虑那个英雄能觉醒的能力. 比如冰女, 觉醒4可以觉醒的能力使得她所有的技能的冷却大量降低, 极大的增强了她控场能力以及AOE伤害. 冰女的觉醒6使得她普通攻击和多达三个技能都能同时攻击多个敌人, 使得她的伤害直接乘于三倍. 作为冰女的反向例子, 小丑即使觉醒5的能力和雷夫搭配也得到不了质变, 而小丑的觉醒6更是大后期的锦标赛才能用到, 而且往往在小丑用的时候要搭配其他强大的英雄. 冰女则不同, 早期如果你拥有一个6阶段的冰女你在低级锦标赛的竞争力将会相对来说很强大, 同时推图的速度也不是6阶段小丑能比的. 而有的一些觉醒是比较单纯考虑到锦标赛的用途然后同时用于凑觉醒等级的目地. 比如说烈焰神龙, 烈焰神龙在推图中期时候的功能性基本是1星, 而冰女推图功能性则是四星, 冰女是一个可以从图2用到图6的英雄. 烈焰神龙四星的觉醒能力是降低飞行敌人的速度, 这在一些一大群一大群飞行敌人来袭的关将非常有帮助. (有种思路就是觉醒神龙到4也将会在锦标赛有一定的竞争力）
必须避免的四个英雄（石头人, 猪男, 以及蝎子以及现在唯一能买到的普通形态康妮, 不知道是哪几个的请去看英雄介绍）
英雄介绍里的康妮是以前某次的活动才能得到的, 被称为亡灵法师康妮, 是可以觉醒6的. 而目前店铺里的康妮只能觉醒1的颓废松鼠. 两者的头像就不一样, 详情可以对比英雄介绍里亡灵康妮的皮肤和现在游戏里康妮的皮肤. 目前官方还没有对于康妮这个英雄有任何的回应, 如果又有亡灵康妮的活动, 到时群里公告一定会通知的.
游戏中要避免购买1500的英雄, 等到玩家要把该觉醒的强力英雄先觉醒了再考虑.
建议新手去游戏里把每个英雄能觉醒的能力都看下, 对英雄的觉醒能力有一定的了解.
菲即使是觉醒6依然很弱.
低阶段的英雄需要的觉醒碎片少, 所以觉醒的能更快, 很多玩家想早点解锁下一个世界地图的话可以选择觉醒菲.(不建议）

3.接下来是笔者推荐的解锁地图的思路了
3.1 解锁地图思路的几个重点
第一个购买的英雄必须是冰女（小丑和冰女都是3000钻）
冰女最好觉醒到6
圣水最好不要投资在菲和骑士身上. 
圣水只在需要的时候的投资, 其他时候请不要使用圣水

解锁世界3到5必须用觉醒以及购买的英雄的觉醒等级总和来解锁
解锁世界6用锦标赛币（够钻石的话也可以买英雄来再次用觉醒等级解锁）
解锁世界4必须有钻石买紫龙（7500钻）<游戏里推图速度的第一个起飞点>
解锁世界5必须买雷夫（7500钻）<游戏里推图速度的第二个起飞点>
觉醒已有的英雄一定会比买英雄便宜. 建议不够钻石买关键英雄的玩家觉醒后期用途较大的英雄得以攒下钻石
必须从游戏一开始就练习如何打锦标赛（假如你想最快速度的推到6图, 否则的话就慢慢来）
钻石除了购买英雄和觉醒外不能用在其他地方（除了花费350钻升级圣水工厂到3）.
注: 目前看来买紫龙和雷夫的所有玩家就没有后悔过的, 而购买不必要英雄的玩家或多或少都后悔过.
3.2 解锁地图思路详细流程
等够350钻的时候把泉水升级到3级
记住要把每天的登录等等的奖励全部领取

玩家确保能简单拿到的钻石全部已经拿到.（详情请看本篇攻略第二部分关于如何获取钻石）
解锁第三世界需要5个觉醒等级（最佳选择: 冰女3, 骑士2. 解锁地图更快的选择: 冰女2 骑士2 菲1）在这之后也一直打冰女的觉醒碎
有卡在某关的可以询问群里的人
世界上2到3的这段旅程攒够钱的话可以先买小丑然后觉醒冰女（买小丑后用圣水升级到12）（现在的队伍是骑士,小丑, 冰女）

解锁第五个地图需要22个觉醒等级(凤凰5, 冰女6, 紫龙5, 骑士3, 小丑3）（剩下的钻确保够7500钻或者说6500钻, 到时候通关130之后可以马上购买雷夫.）
通关130所用的英雄队伍（紫龙,冰女 ,小丑或者骑士）通关购买雷夫后, 用圣水把雷夫升级到20级.
锦标赛是从游戏最开始一直打起, 希望现在已经够两百锦标赛币了, 没有的话就是时候慢下来步伐打锦标赛的时候了.
开启世界6, 用圣水把凤凰升级到20级左右（打世界6的英雄队伍是凤凰5, 雷夫5以及冰女6 （到这里为止你也不是新玩家了, 所以该用哪个英雄你应该知道了, 上面只是建议而已））

4.一些关于锦标赛的通常信息

锦标赛是15名到50名玩家按照先来后到的顺序分配到一个小组 低级别的竞赛是取前三名晋级到下一等级的锦标赛. 分别有青铜,白银,黄金,白金,钻石,大师,传奇. 传奇等级的锦标赛拿到第一后下周继续回到传奇锦标赛. 似乎是每三个月一个赛季,希望知道的人来向我更正下. 每个新的赛季开始, 打到金锦标赛以上的玩家将重新回到金竞标赛. 以下是各级锦标赛的各种名次和其相对应的奖励:

 4.1以下是一些关于锦标赛的小技巧：
1.建议玩家完成世界2就开始打锦标赛并且不断的练习
2.青铜等级的锦标赛的使用道具建议可以用合成的钱袋和霜冻药水.（青铜锦标赛无法使用流星, 双倍霜冻药水以及陨石） 道具详细信息请看游戏里的铁匠铺（世界2开启）
3.银等级的锦标赛的使用可以考虑用混合1个金袋三个流星的道具或者4个流星（银等级锦标赛无法使用陨石）
4.金及以上等级的锦标赛使用的道具是4个陨石
5.双倍霜冻药剂理论上是不需要的, 所以我的意见是除了少量合成金袋外, 唯一需要的就是流星和陨石的合成.
6.保证自己晋级白银, 黄金以及白金的时候是以第一名晋级的, 建议新手玩家不要太快晋级到黄金, 因为到时候连低保（也就是4到10名都是所谓的低保）都拿不到.
7.坚持每天用完免费的打锦标赛的机会, 确定自己的塔防思路, 然后最后一天最好在锦标赛倒数两小时左右的时候开始打锦标赛, 三次机会, 道具可以全用, 一般是4个流星或者4个陨石每一次机会.
8.锦标赛可以点击段位去看你这个段位最好的英雄队列, 这样你每次就可以学到你这个段位以及这个地图最好的组队思路.
9.锦标赛看其他玩家英雄队列的时候主要注意的点是这个英雄的边框颜色. 英雄的边框颜色对应的是他们的觉醒等级. 分别是青铜是铜色是觉醒0， 银色是阶段1-2， 金色是阶段3-5， 紫色是阶段6. 这样看别人的队列就大概知道别人队列的英雄是觉醒几了. 
10.兵营可以拆了再建用来复活士兵， 其他建筑一样, 比如魔法塔可以再次激活瞬移走敌人.
11.尽量让下一波的敌人提早来可以得到更多的金币.
12.用完英雄的技能马上叫下一波敌人可以连续再用技能一次.
13.尽量把英雄放在一个节点上让英雄不需要过多的移动.
14.蝎子和猪男是被祝福了之后还是能被取代的英雄, 谨慎使用, 尤其蝎子是就算祝福了也是被扔仓库的. 猪男有兴趣的玩家可以研究下, 不一定可以有很好的配合，所以猪男被祝福是有可能上场的, 尤其的搭配亡灵法师形态的康妮, 因为猪男的特长是加强召唤生物（注：康妮亡灵法师形态仅在活动中出现过, 新玩家无法获取, 目前能获取的形态是不能觉醒的形态， 完全没有拿的价值）


 4.2 一些锦标赛队伍搭配的最基本思路
不能是三个坦克,因为那个就没输出了. 不能是三个辅助因为辅助了之后也是没人当那个输出了. 当然也不能是三个控场, 因为那样就没有输出环境了.
一些强力英雄加上祝福英雄的万金油队伍, 往往都给能给予不错的名次. 雷打, 鲤助还有祝福英雄.
祝福英雄是菲的时候可以搭配火焰神龙和波尔顿（菲必须是觉醒6）
冰女是祝福英雄的时候6阶段的嫣可以上场继续加强冰女
神龙是祝福英雄的时候紫龙可以上场加强神龙
